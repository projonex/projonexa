# GitHub Labels — Quick Setup

Create these labels before opening issues from [GITHUB_ISSUES_BACKLOG.md](./GITHUB_ISSUES_BACKLOG.md).

**Path:** Repository → **Issues** → **Labels** → **New label**

## Required (automation)

| Label | Color | Description |
|-------|-------|-------------|
| `dependencies` | `#0366d6` | Dependabot dependency updates |

## Standard (from repository setup)

| Label | Color | Description |
|-------|-------|-------------|
| `bug` | `#d73a4a` | Something is broken |
| `enhancement` | `#a2eeef` | New feature or improvement |
| `documentation` | `#0075ca` | Documentation only |
| `question` | `#d876e3` | Question or discussion |
| `needs-triage` | `#fbca04` | Needs review |
| `good first issue` | `#7057ff` | Good for newcomers |
| `help wanted` | `#008672` | Extra attention welcome |
| `github-actions` | `#000000` | CI/CD workflows |
