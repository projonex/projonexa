# `src/lib`

Shared utilities used across the app (not React components).

| Path | Purpose |
|------|---------|
| `seo/` | Metadata, JSON-LD, sitemap URLs, share images — import from `@/lib/seo` or `@/lib/seo/...` |
| `navigation.ts` | Active nav link helper |
| `motion.ts` | Shared Framer Motion easing |
| `referralCode.ts` | Affiliate referral code helpers |
| `techCloud.ts` | Technology icon cloud config |
| `theme.ts` | Re-exports design tokens from `src/theme/` |
